# Popover

## Description

The Popover is a wrapper intended to have other react components passed in as children.

## Preview

      <PopoverContent radius="small">
        <div style={{padding: '8px'}}>This is my popoverContent</div>
      </PopoverContent>
    </Popover>
  </Story>
</Preview>
