# List

## Description

The List is a wrapper intended to have the ListItem react components passed in as children.

## Preview

<Preview>
  <Story name="Default">
    <List>
      <ListItem>Non-clicky item</ListItem>
      <ListItem onClick={() => alert('Oi, don\'t touch me!')}>Clicky item</ListItem>
      <ListItem disabled onClick={() => alert('Nah na na na, can\'t touch this')}>Disabled item</ListItem>
    </List>
  </Story>
</Preview>
