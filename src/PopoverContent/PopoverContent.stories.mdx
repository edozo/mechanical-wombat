# PopoverContent

## Description

The Popover Content is a wrapper intended to have other react components passed in as children.

## Preview

<Preview>
  <Story name="Default">
    <PopoverContent borderRadius="small">
      <div style={{padding: '8px'}}>This is my popoverContent</div>
    </PopoverContent>
  </Story>
</Preview>

## Variations

<Story name="List">
  <PopoverContent>
    <List>
      <ListItem>Non-clicky item</ListItem>
      <ListItem onClick={() => alert('Oi, don\'t touch me!')}>Clicky item</ListItem>
      <ListItem disabled onClick={() => alert('I am invisible')}>Disabled item</ListItem>
    </List>
  </PopoverContent>
</Story>
