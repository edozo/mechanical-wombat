# Popover

## Description

The Popover is a wrapper intended to have other react components passed in as children.

## Preview

<Preview>
  <Story name="Default">
    <Popover radius="small">
      <div style={{padding: '8px'}}>This is my popover</div>
    </Popover>
  </Story>
</Preview>

## Variations

<Story name="List">
  <Popover>
    <List>

      <ListItem onClick={() => alert('Oi, don\'t touch me!')}>Clicky item</ListItem>
      <ListItem disabled onClick={() => alert('I am invisible')}>Disabled item</ListItem>
    </List>
  </Popover>
</Story>
